# 🎤 Interview Coach AI

Aplikasi latihan wawancara kerja berbasis AI — gratis, langsung di browser, tanpa daftar akun.

---

## ✨ Apa ini?

**Interview Coach AI** adalah teman latihan wawancara yang bisa kamu ajak ngobrol kapan saja.

Kamu isi profil, upload CV (opsional), lalu AI akan:
- 🔮 **Memprediksi** pertanyaan yang kemungkinan besar muncul di interview kamu
- 💬 **Mensimulasikan** sesi wawancara nyata — tanya jawab interaktif
- 🎯 **Memberi feedback** setelah tiap jawaban: apa yang bagus, apa yang bisa diperbaiki
- ✨ **Menuliskan ulang** jawabanmu versi lebih baik sebagai contoh
- 🔊 **Membacakan** pertanyaan & feedback dengan suara (Text-to-Speech)
- 🎙️ **Menerima jawaban lewat suara** — bicara langsung, tidak perlu mengetik

---

## 🚀 Cara Pakai

### 1. Siapkan API Key (gratis)

Aplikasi ini menggunakan AI dari Google Gemini. API key-nya **gratis**:

1. Buka [aistudio.google.com](https://aistudio.google.com)
2. Login dengan akun Google
3. Klik **"Get API Key"** → **"Create API Key"**
4. Copy key-nya

> 💡 Tidak mau pakai Gemini? Bisa juga pakai api lain bebeas pilih

### 2. Buka Aplikasi

Kunjungi aplikasi dan masukan api mu.

### 3. Isi Profil

- Cukup isi Nama & Posisi yang dilamar sudah bisa langsung wawancara
- Tambahkan pendidikan, pengalaman, skills & nama perusahaan untuk hasil yang lebih akurat
- Pilih bahasa: English, Bahasa Indonesia, atau bahsa lainya
- Bisa upload CV / job description kalau ada (PDF, gambar, txt)

### 4. Lihat Prediksi Pertanyaan

Klik **"Lihat Prediksi Pertanyaan"** — AI akan menampilkan pertanyaan yang kemungkinan besar ditanyakan, lengkap dengan tips dan contoh jawaban.

### 5. Mulai Sesi Wawancara

Klik **"Mulai Wawancara"** — AI akan menyapa dan memandu sesi latihan:
- AI tanya satu per satu
- Kamu jawab (ketik atau pakai mic 🎙️)
- Dapat feedback langsung setiap selesai menjawab
- Bisa minta contoh jawaban kalau buntu: *"kasih contoh jawaban"*

## 🌐 Bahasa yang Didukung

| Setting | Pertanyaan AI | Feedback |
|---|---|---|
| **Default** | Inggris + terjemahan | Indonesia|
| **English** | Inggris | Inggris (koreksi grammar) |
| **Bahasa Indonesia** | Indonesia | Indonesia |
| **Bahasa lain(isi sendiri)** | Aksara bahasa asing yang dipilih | bisa pakai bahasa yang kamu mau|

## 🔒 Privasi

- ✅ **Tidak ada akun** — tidak perlu daftar atau login
- ✅ **API key tersimpan di browser kamu** — tidak dikirim ke server kami
- ✅ **Tidak ada database** — semua data ada di browser kamu sendiri
- ⚠️ Data kamu dikirim langsung ke provider AI (Gemini/OpenRouter) sesuai API key yang kamu pakai
